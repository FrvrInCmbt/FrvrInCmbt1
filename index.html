List of terminal comands that we've been looking on the lesson

  • git status - shows changes , staged changes , unstaged , modified files etc.
  • git checkout "here shoudl be name of branch that you want to switch" - switching beetween branches
  • git commit -m "here should be your message" - making commit with message specified in quotes
  • git push - sends a local branch to a remote repository
  • git add - adds specified file as staged changes before commit , to allow Git track changes in specified file
  • git clone - set up connect by HTTP or SSH key between local machine and remote repository
